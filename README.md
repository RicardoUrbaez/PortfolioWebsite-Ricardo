## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** JavaScript
- **Package Manager:** Npm

### Installation

Build PortfolioWebsite-Ricardo from the source and install dependencies:

1. **Clone the repository:**

    ```sh
    ❯ git clone https://github.com/RicardoUrbaez/PortfolioWebsite-Ricardo
    ```

2. **Navigate to the project directory:**

    ```sh
    ❯ cd PortfolioWebsite-Ricardo
    ```

3. **Install the dependencies:**

**Using [npm](https://www.npmjs.com/):**

```sh
❯ npm install
```

### Usage

Run the project with:

**Using [npm](https://www.npmjs.com/):**

```sh
npm start
```

### Testing

Portfoliowebsite-ricardo uses the {__test_framework__} test framework. Run the test suite with:

**Using [npm](https://www.npmjs.com/):**

```sh
npm test
```
